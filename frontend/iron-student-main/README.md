# ПК-Майстерня — Візуальний прототип

Освітня веб-платформа для навчання студентів збірки комп'ютерів з кіберпанк-дизайном.

---

## 📦 Структура проекту

```
iron-student/
├── index.html          # Головний HTML-файл (чиста структура)
├── styles.css          # Всі стилі CSS
├── prototype.html      # Старий файл (можна видалити)
└── prototype.css       # Старий файл (можна видалити)
```

---

## 🎨 Опис файлів

### `index.html` (≈570 рядків)
**Призначення**: Основна HTML-структура без вбудованого CSS

**Що містить**:
- **Header** (рядки 19-31): Логотип + навігація (Дашборд, Тест, Гра, Адмін)
- **Дашборд** (рядки 37-103): Hero-секція з прогресом + 4 карточки (Тест, Гра, Сертифікати, Методички)
- **Тест** (рядки 108-150): Сторінка з питаннями, таймером, варіантами відповідей
- **Гра** (рядки 155-262): Симулятор збірки ПК з 3 панелями (чеклист, схема ПК, деталі)
- **Адмін-панель** (рядки 267-333): Таблиця студентів з прогресом і ролями
- **Модальне вікно** (рядки 339-361): Результат тесту після завершення
- **JavaScript** (рядки 368-566): Мінімальна логіка для демонстрації (переключення сторінок, навігація по тесту, анімації)

**Ключові особливості**:
- Українська мова інтерфейсу
- Семантична верстка (section, nav, main, header)
- Data-атрибути для JS (`data-page`)
- Font Awesome 6 для іконок

---

### `styles.css` (≈1150 рядків)
**Призначення**: Повний стильовий файл з кіберпанк-темою

**Структура** (по секціях):

#### 1. **Базові стилі** (рядки 1-40)
- Імпорт шрифтів: `Orbitron` (заголовки), `Inter` (текст)
- Background: темно-синій `#0A0C10` з радіальною сіткою
- Контейнер для всієї сторінки

#### 2. **Анімації** (рядки 42-66)
- `fadeIn`: плавна поява елементів
- `neon-pulse`: неоновий пульсуючий ефект для заголовків
- `blink-green` / `blink-red`: анімація для правильних/неправильних відповідей

#### 3. **Header / Навігація** (рядки 68-110)
- Логотип з неоновим світінням (`#00D4FF`, `#7B2CBF`)
- Кнопки навігації зі скломорфізмом (backdrop-filter)
- Активний стан з підсвічуванням

#### 4. **Загальні стилі сторінок** (рядки 112-123)
- `.page` з `display: none` для переключення
- `.page.active` для відображення

#### 5. **Дашборд студента** (рядки 125-300)
- Hero-секція з привітанням і бейджем
- Прогрес-бар з градієнтом
- Сітка 2×2 з карточками
- Hover-ефекти (підйом, збільшення, світіння)

#### 6. **Тест** (рядки 302-450)
- Контейнер з таймером і прогрес-баром
- Placeholder для зображень компонентів
- Кнопки відповідей з анімацією при виборі
- Зелений/червоний фідбек

#### 7. **Гра (симулятор збірки ПК)** (рядки 452-750)
- Grid-layout: 3 колонки + нижня панель
- **Ліва панель**: чеклист завдань, кнопки керування (W/A/S/D/E/F)
- **Центр**: схема ПК з 6 слотами (CPU, RAM, GPU, MB, PSU, SSD)
- **Права панель**: список доступних деталей з цінами
- **Низ**: підсумок замовлення та кнопки дій

#### 8. **Адмін-панель** (рядки 752-900)
- Таблиця студентів з прогресом
- Бейджі ролей (Студент / Адмін)
- Кнопки експорту даних

#### 9. **Модальне вікно** (рядки 902-1050)
- Overlay з розмиттям
- Анімація fadeIn
- Кнопка закриття (×)
- Іконка трофею для успіху

## 🔧 Технології

- **HTML5**: Семантична структура, data-атрибути
- **CSS3**: Grid, Flexbox, backdrop-filter, animations, gradients
- **JavaScript (мінімальний)**: Vanilla JS для демонстрації
  - Переключення сторінок
  - Навігація по питаннях тесту
  - Показ модального вікна
  - Анімація прогрес-барів
- **Шрифти**: Google Fonts (Orbitron, Inter)
- **Іконки**: Font Awesome 6 (CDN)

## 🚀 Як користуватися

### 1. Відкрити проект
Просто відкрийте `index.html` у браузері:
```bash
open index.html  # macOS
```
або перетягніть файл у вікно браузера.

### 2. Навігація
- Перемикайтесь між сторінками через навігацію у хедері
- **Дашборд**: Огляд прогресу студента
- **Тест**: Відповідайте на 3 питання про комплектуючі
- **Гра**: Симулятор збірки ПК (статичний макет)
- **Адмін**: Панель адміністратора з таблицею студентів

### 3. Інтерактив
- Клік на відповідь у тесті → зелений/червоний фідбек
- Кнопка "Завершити" → модальне вікно з результатом
- Hover-ефекти на карточках, кнопках, слотах

## 🎯 Призначення

**Це візуальний прототип** для демонстрації дизайну платформи.
